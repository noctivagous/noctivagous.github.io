Keyboard Shortcuts

Alt + Shift + B Toggle KeyPilot on/off
Alt + Shift + P Toggle debug mode
F Click elements (when KeyPilot is active)
H Highlight text (when KeyPilot is active)
D Delete elements (when KeyPilot is active)
ESC Exit current mode
