Positioning Windows And UI Elements Through Guidelines

Take a box and apply the function.

var box = rootViewport //(800 x 200)

// this returns three boxes flowing horizontally
horizontalSubdivisionBoxes = box.subdivideHorizontally(3);

// this returns four boxes flowing vertically,
// not connected to the horizontal boxes.
verticalSubdivisionBoxes = box.subdivideVertically(4);


// POINT BASED ALIGNMENT AND SIZING
// Make the array to get intersection points
var intersectionArray = [horizontalSubdivisionBoxes, verticalSubdivisionBoxes];

// Get the intersection points from two boxes.
// Returned points flow from left to right, row major order.
var intersectionPointsArray = intersectionPoints(intersectionArray);

var topLeftPoint =intersectionPointsArray[0];

// center an existing box, box1 on the point
box1.centerOnPoint(topLeftPoint);
