🎮 How to use this demo

1. Press ↓ ArrowDown. This turns on the DOWN scroll mode

2. Move your mouse up and down (back and forth) — the page scrolls continuously in the direction of the mode.

3. The scroll amount matches the distance you move the mouse.

4. Press any key (Space, Ctrl, Esc, letters, etc.) to instantly exit the mode.

Welcome to the demo document

This is a long scrollable page designed to showcase the nct-ui ScrollUpOrDownWithAnyMouseMovement-Mode.

Imagine you are reading a massive research paper, a long technical specification, or a very tall webpage. Normally you would use the scroll wheel, scrollbar, or Page Down. With this mode you simply press one key and then move your mouse vertically — no more reaching for the wheel, no more hitting the edge of your mousepad.

Why back-and-forth works so well

  • Mouse movement is measured by distance only (absolute value).
  • Direction is locked to the mode you chose (↑ or ↓).
  • Move the mouse down → scroll down.
  • Move the mouse back up → still scrolls down (by the same distance).
  • Repeat the motion as long as you want. No need to lift the mouse or reset position.

Section 1 — Feature Details

Once activated, the system captures every pixel of vertical mouse movement and converts it into smooth document scrolling in the fixed direction. Horizontal movement is completely ignored.

This mode is perfect for users who want to keep their hand on the mouse while using a special hardware key on the left side of the keyboard.

Section 2 — Real-world use cases

• Reading long articles or books in browser
• Browsing technical documentation
• Reviewing code repositories
• Infinite social feeds or timelines
• Accessibility mode for users with limited wrist movement
